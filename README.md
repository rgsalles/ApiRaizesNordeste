# Minha API

API desenvolvida com ASP.NET Core e banco de dados SQL Server.
Este projeto visa atender as necessidades da rede Raízes do Nordeste.
Um sistema desenvolvido para atendimento de clientes, funcionários e gestores. Permitindo a gestão de pedidos, cardápios e relatórios operacionais.

## Tecnologias

- .NET
- ASP.NET Core
- OpenAPI
- SQL Server
- Swagger

## Como executar, escolha um dos ambientes

```bash
dotnet restore
dotnet run --launch-profile Development
dotnet run --launch-profile Staging
dotnet run --launch-profile Production
